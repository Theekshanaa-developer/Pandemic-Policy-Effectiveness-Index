# Lag Optimization Using CCF — Module 3

This module estimates the delay (lag) between government policies
(Stringency Index) and epidemiological outcomes (COVID-19 cases) using
Cross-Correlation Function (CCF).

The lag is a key component of PPEI and reflects how quickly policies
produce measurable effects.

## Outputs
- `lag_results.csv` — lag days for each country
- `lag_distribution.png` — histogram of lags
- `india_ccf_plot.png` — detailed CCF curve for India
- `wave_lag_summary.csv` — if wave-lag extension is added
